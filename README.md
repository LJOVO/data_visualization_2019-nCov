# data visualization - 2019-nCov Epidemic in China  
## 2019-nCov中国疫情数据可视化  
>病例数据来自[丁香园] (https://3g.dxy.cn/newh5/view/pneumonia)  
>>environment:  
>>Python 3.7   
>>pyecharts 1.6.2   
>>BeautifulSoup4 4.8.2

大致思路:  
利用BeautifulSoup4对上述URL进行解析, 经过处理后获得当前病例数据.  
再利用pyecharts v1 进行数据可视化. 并生成相应的html文件.
